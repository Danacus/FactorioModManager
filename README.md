Factorio Mod Manager!
===================
----------
A mod manager for Factorio

----------

Features
-------------
 - Automatically download and update mods from the Mod Portal
 - Sort mods by downloads or in alfabetical order
 - Search your favorite mods
 - Create your own modpacks
 - Import modpacks
